## 三 解题思路



二分查找：

```js
const findMin = (nums) => {
  let left = 0, right = nums.length - 1;
  while (nums[left] > nums[right]) {
    const middle = Math.floor((left + right) / 2);
    if (nums[middle] > nums[right]) {
      left = middle + 1;
    } else if (nums[middle] < nums[right]) {
      right = middle;
    }
  }

  return nums[left];
};
```
