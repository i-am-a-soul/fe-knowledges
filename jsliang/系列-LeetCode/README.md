## <a name="chapter-two" id="chapter-two">二 前言</a>



LeetCode 是个题库，里面有很编程多面试的题目，可以在线编译运行。

在这里，你可以使用 C、C++、Java、Python、PHP、JavaScript 共 14 种编程语言进行解题。

在这里，你可以接触到许许多多算法相关的知识，并且提升自己的职业竞争能力，在面试的时候秀一秀自己的代码。
