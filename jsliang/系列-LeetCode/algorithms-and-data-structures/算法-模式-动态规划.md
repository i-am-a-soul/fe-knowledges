## <a name="chapter-two" id="chapter-two">二 前言</a>



先看扑克牌题目：

```
有 54 张扑克牌，两人轮流从中抽取牌，
每人至少抽 1 张，最多抽取 3 张，
谁抽取到最后 1 张就算赢。
```

先抽取 2 张，以后每次根据对手抽的张数 n(1≤n≤3)，抽取 4-n 张，即可保证一定获胜。
