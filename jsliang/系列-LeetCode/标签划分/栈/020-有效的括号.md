给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，
判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"
输出: true
示例 2:

输入: "()[]{}"

示例 4:

输入: "([)]"

示例 5:

## 三 解题思路



栈：

```js
/**
 * @name 平衡圆括号
 * @param {string} s
 * @return {boolean}
 */
const isValid = (s) => {
  const stack = [];
  for (let i = 0; i < s.length; i++) {
    if (
      (s[i] === ')' && stack[stack.length - 1] === '(')
      || (s[i] === ']' && stack[stack.length - 1] === '[')
      || (s[i] === '}' && stack[stack.length - 1] === '{')
    ) {
      stack.pop();
    } else {
      stack.push(s[i]);
    }
  }
  console.log(stack);
  return stack.length === 0;
};
```
