## 三 解题思路



Set 集合：

```js
/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @return {number[]}
 */
const intersection = (nums1, nums2) => {
  // 1. 设置集合
  const set = new Set();

  // 2. 遍历数组一，判断数组二是否包含当中元素
  for (let i = 0; i < nums1.length; i++) {
    if (nums2.includes(nums1[i])) {
      set.add(nums1[i]);
    }
  }

  // 3. 因为 set 添加的已经是去重的，所以返回数组格式即可
  return [...set];
};

console.log(intersection([4, 9, 5, 9], [9, 4, 9, 8, 4])); // [4, 9]
```
