把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。

输入一个递增排序的数组的一个旋转，输出旋转数组的最小元素。

## 三 解题思路



```js
const minArray = (numbers) => {
  let i = 0; j = numbers.length - 1;
  while (i < j) {
    const m = Math.floor((i + j) / 2);
    if (numbers[m] > numbers[j]) {
      i = m + 1;
    } else if (numbers[m] < numbers[j]) {
      j = m;
    } else {
      j--;
    }
  }
  return numbers[i];
};
```
