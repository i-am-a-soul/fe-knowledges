什么是 GitHub Pages？

GitHub Pages 是一个通过 GitHub 项目/仓库 进行部署的，以 `用户名.github.io` 的形式生成用户的托管平台。

通过 GitHub Pages，用户只需要把自己的内容在本地编辑好，然后推送到 GitHub 仓库即可。
