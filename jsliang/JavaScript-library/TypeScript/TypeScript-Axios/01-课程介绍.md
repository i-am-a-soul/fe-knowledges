## 二 前言



* 什么是 TypeScript？

1. JavaScript 的增强
2. 未来前端开发趋势
3. 技术转型的趋势
4. 提升个人能力和竞争力

TypeScript 是 JavaScript 的超集，它可以编译成纯 JavaScript。

TypeScript 可以在任何浏览器、任何计算机和任何操作系统上运行，并且是开源的。

* 官方文档：https://www.tslang.cn/index.html

## 三 课程概述



* 这门课程做什么？

使用 TypeScript 从零实现一个 Axios 库。

* 这门课程有哪些内容？

1. TypeScript 基础知识
2. TS-Axios 开发实战
3. 项目的测试、构建与发布
